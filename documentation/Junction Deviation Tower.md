## Printing Conditions
     - Nozzle Size : 0.4
     - Layer Height : 0.2
     - Initial Layer Height : 0.2
     - Line Width : 0.4
     - Wall Line Count : 2
     - Top Thickness : 0 mm
     - Bottom Thickness : 0.8 mm
     - Infill Density : 0%
